使用 Hexo 搭建个人博客

一、安装环境

  • Node.js
  • Git
  • Hexo

1、安装 Node.js

【Node.js 下载链接】

2、安装 Git

【Git下载链接】

3、查看安装结果

  • 打开命令行

    win + R 输入 cmd / ** win10 搜索应用 命令 **/ 任意目录 shift +鼠标右键

  • 输入命令

    node -v

    git --version

  • 安装成功即可看见相对应软件的版本号

4、安装 hexo

  • 打开 Git Bash：任意文件夹右键 -> Git Bash Here

  • 安装 Hexo ：输入命令:

    $ npm install hexo-cli -g

  • 安装完成后，输入 hexo -v，如出现 hexo 的版本号则安装成功

二、初始化博客

  1. 创建 hexo 目录：新建一个文件夹，个人博客文件都将存放在此

    • 在目录下右键打开 Git Bash Here 。

    • 输入命令 : hexo init

    • 看到 INFO Start blogging with Hexo! 表明初始化成功

    • 安装依赖：npm install

三、修改博客主题

我使用的是 hexo-theme-beep这款主题， 其中文文档中有详细的步骤

记录一下我去掉的主题功能和修改了哪些部分

~ 代指博客的根目录

  • ~/_config.yml

    • title: Ag-H2O
    • author: Ag-H2O
    • theme: beep
    • deploy:
      type: git
      repo:
      github: git@github.com:Ag-H2O/Ag-H2O.github.io.git,master

  • /themes/beep/_config.yml

    • 侧边栏的图标与超链

  • themes/beep/source/js/src/custom.js 去掉了几个 js

  • themes\beep\source\css\_custom\custom.styl

    • 修改爱心颜色
    • 去除鼠标样式
    • 修改文章标题样式
    • 修改 /* 站点标题 */ 的阴影

  • 修改网站图标

    • 文件地址 ~\themes\beep\source 的两个 ico 文件
      在线图片转icon格式 – 图片转换成icon

  • 修改侧边栏头像 ：~\ themes\beep\source\images\avatar.jpg

  • 修改右上角 github 的超链接 ~\themes\beep\layout_layout.swig
    搜索 github.com

四、本地查看博客效果

  • 输入命令 ： hexo g

    `hexo s`

  • 根据提示在浏览器中输入http://localhost:4000/

写第一篇博客

Hexo分为 post 和 draft 两种，其中post 存放在 source/_posts 目录下，draft 存放在 source/_drafts 目录下，区别在于前者会被发布到博客，而后者不会被发布。

第一篇文章

文章中的图片

在 source/ 目录下新建一个文件夹，比如 images；在该文件夹下存放各种图片资源，比如 test.jpg；
新建一篇文章，在文章中使用markdown的插入图片语法：
![test](/images/test.jpg)
这里使用的是绝对路径，如果使用相对路径比如images/test.jpg可能会无法访问到该图片资源。建议在博客或者配置中都使用绝对路径。
